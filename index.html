01

El Antídoto

Sin vocabulario de RevOps, tu asistente de IA hace análisis genérico. No sabe qué es un coverage ratio, cómo ajustar un forecast por win rate real, ni cómo calcular el health score de una cuenta. Fluxi inyecta ese vocabulario: frameworks curados, anti-patrones y criterios de decisión listos para usar.

¿Falta un framework? Sugerilo →

  • Gestión de Pipeline

    Ratios de cobertura, velocity, conversión por etapa y deals estancados. Benchmarks B2B SaaS incluidos.

  • Revenue Forecasting

    Pipeline ponderado ajustado por win rate real. Tres escenarios: peor caso, base y mejor caso.

  • Lead Scoring

    Modelo multicritério Fit × Engagement × Timing. Umbrales de acción por tier para SDRs.

  • Anti-Patrones

    Los 7 errores más comunes en RevOps que el AI aprende a detectar y evitar automáticamente.

03

Comandos

Siete comandos que cubren todo el stack de inteligencia de revenue.

Comando Qué hace
/icp-market

Define tu Perfil de Cliente Ideal a partir de deals ganados y estima TAM / SAM / SOM. Scorecard ICP con criterios de calificación y señales de compra.

/pipeline-health

Coverage ratio, velocity, conversión por etapa, deals estancados y alertas por rep. Acciones priorizadas para las próximas 48–72 horas.

/forecast

Pronóstico con tres escenarios ajustado por win rate real del equipo. Gap analysis y plan para cerrar el shortfall.

/churn-risk

Health scores por cuenta en 5 dimensiones. Señales de alerta temprana y playbooks de retención priorizados por ARR en riesgo.

/funnel-diagnosis

Conversión de punta a punta, time-in-stage, análisis de loss reasons y causa raíz por etapa.

/lead-scoring

Modelo Fit × Engagement × Timing. Tiers A–D con SLA de acción. Calibrado con datos históricos de deals ganados.

/deal-review

Revisión MEDDPICC con score de completitud, probabilidad ajustada por stakeholder y próximos pasos críticos en 72 horas.

04

Instalación

Un comando. Todos los providers. Auto-detecta tu AI harness.

$ npx skills add fluxilabs/fluxi

Auto-detecta tu AI harness e instala los archivos en la ubicación correcta.

# Clonar el repositorio
$ git clone https://github.com/fluxilabs/fluxi

# Copiar al directorio global de Claude
$ cp -r dist/claude-code/.claude ~/.claude

También podés copiar a /tu-proyecto/.claude para instalación por proyecto.

# Cursor
$ cp -r dist/cursor/.cursor /tu-proyecto/
# Gemini CLI
$ cp -r dist/gemini/.gemini ~/
# Codex CLI
$ cp -r dist/codex/.codex ~/
# Kiro · VS Code Copilot
$ cp -r dist/agents/.agents /tu-proyecto/

O descargá el ZIP desde GitHub Releases.
